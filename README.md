#  Password Generator Web App

A modern, customizable **Password Generator** built with **React.js** and **Tailwind CSS**.  
It allows users to generate secure passwords based on their chosen criteria and provides a clean, responsive user interface.

---

## Features

### Current Features
- Generate random passwords
- Adjustable password length
- Copy to clipboard
- Toggle options:
  - Include Uppercase letters
  - Include Lowercase letters
  - Include Numbers
  - Include Special Characters
- Real-time password display
- Responsive UI using Tailwind CSS

---

##  Upcoming Features

- **Password Strength Meter**  
  Shows password strength using entropy & pattern rules.

- **Password History (localStorage)**  
  Saves previously generated passwords.

- **Dark / Light Theme Toggle**

- **Passphrase Generator**  
  Creates long, secure word-based passphrases.

- **Offline Breach Check**  
  Detects if the password is used in common leaked password lists.
